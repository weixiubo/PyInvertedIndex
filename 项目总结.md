## 🎓 技术要点

### 核心算法
1. **倒排索引构建**
   - 遍历文档，提取词项
   - 记录词项位置
   - 构建倒排列表

2. **短语查询算法**
   - 获取第一个词的文档列表
   - 验证后续词的连续性
   - 基于位置信息判断

3. **布尔查询**
   - AND: 集合交集
   - OR: 集合并集
   - NOT: 集合差集

### 数据结构
```python
{
    "词项": {
        "文档ID": [位置1, 位置2, ...],
        ...
    },
    ...
}
```
